# Song Clustering ML
## Overview
This project applies machine learning clustering techniques to analyze and group songs based on their audio features and characteristics.

## Features
- Data preprocessing and feature extraction
- Multiple clustering algorithms (K-means, DBSCAN, Hierarchical)
- Visualization of clusters
- Model evaluation and metrics

## Installation
```bash
git clone <repository-url>
cd Song_Clustering_ML
pip install -r requirements.txt
```
